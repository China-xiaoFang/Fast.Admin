# App.Client 客户端文档

> 🚧 **本文档正在规划中**

## 概述

App.Client 是 Fast.Admin 的客户端应用，基于 React + TypeScript 构建。

## 技术栈

| 技术 | 说明 |
|------|------|
| React | UI 框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |

## 目录结构

```
App.Client/
├── src/
│   ├── components/     # 公共组件
│   ├── pages/          # 页面
│   ├── services/       # API 服务
│   ├── stores/         # 状态管理
│   └── utils/          # 工具函数
├── public/             # 静态资源
└── package.json        # 依赖配置
```
